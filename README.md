## Aura

This repository is the root workspace for the Aura project.

### Repo structure

- **`apps/web/`**: Next.js (App Router + Tailwind) frontend
- **`packages/`**: shared code (UI, utilities, config) — to be added as Aura grows
- **`docs/`**: product/tech documentation
- **`research/`**: PDFs and research material (kept out of the app)

### Requirements

- **Node.js**: `>= 20.9.0` (required by the generated Next.js version)

### Run the web app

```bash
cd apps/web
npm install
npm run dev
```
